Univariate Gaussian Density Estimation

Scalar Notation one feature Vector Notation all features at once
Training Mean
\(\mu_j = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m} x_j^{(i)}\)
\(\vec{\mu} = \bigl[\mu_1\;\; \mu_2\;\; \cdots\;\; \mu_n\bigr]\)
Variance
\(\sigma_j^2 = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m} (x_j^{(i)} - \mu_j)^2\)
\(\vec{\sigma}^2 = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m} (\vec{x}^{(i)} - \vec{\mu})^{\odot 2}\)
\( = \bigl[\sigma_1^2\;\; \sigma_2^2\;\; \cdots\;\; \sigma_n^2\bigr]\)
Inference Per-Feature Probability Density
\(p(x_j \mid \mu_j, \sigma_j^2) = \dfrac{1}{\sigma_j \sqrt{2\pi}} \enspace e^{\dfrac{-1}{2} \left(\dfrac{x_j - \mu_j}{\sigma_j}\right)^2}\)
\(p(\vec{x} \mid \vec{\mu}, \vec{\sigma}^2) = (\mathbf{1}_{1 \times n} \oslash \sqrt{2\pi} \,\, \vec{\sigma}) \odot e^{\displaystyle \frac{-1}{2} [(\vec{x} - \vec{\mu}) \oslash \vec{\sigma}]^{\odot 2}}\)
Joint Probability Density
\(p(x) = \displaystyle\prod_{j=1}^n p(x_j \mid \mu_j, \sigma_j^2)\)
\(p(\vec{x}) = \displaystyle\prod p(\vec{x} \mid \vec{\mu}, \vec{\sigma}^2)\)
product of elements in the per-feature density vector
Model /
Prediction
\(\hat{y} = \begin{cases} 1 & \text{if } p(x) < \epsilon \\ 0 & \text{otherwise} \end{cases}\)
\(\hat{y} = \begin{cases} 1 & \text{if } p(\vec{x}) < \epsilon \\ 0 & \text{otherwise} \end{cases}\)
