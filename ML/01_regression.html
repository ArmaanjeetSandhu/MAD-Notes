Cost
\(J(\vec{w}, b) = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m} L\!\left(\hat{y}^{(i)}, y^{(i)}\right)\)
\(J(W, B) = \dfrac{1}{m}\cdot\mathbf{1}\cdot L(\hat{Y}, Y)^{\top}\)
Cost Gradient
\(\dfrac{\partial J}{\partial w_j} = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m}\!\left(\hat{y}^{(i)}-y^{(i)}\right) x_j^{(i)}\)
\(\dfrac{\partial J}{\partial W} = \dfrac{1}{m}\,X\!\left(\hat{Y} - Y\right)^{\!\top}\)
Gradient Descent
Update (weights)
\(w_j := w_j - \alpha\,\dfrac{\partial J}{\partial w_j}\)
\(W := W - \alpha\,\dfrac{\partial J}{\partial W}\)
Cost Gradient
(bias term)
\(\dfrac{\partial J}{\partial b} = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m}\!\left(\hat{y}^{(i)}-y^{(i)}\right)\)
\(\dfrac{\partial J}{\partial B} = \dfrac{1}{m}\cdot\mathbf{1}\cdot\left(\hat{Y}-Y\right)^{\!\top}\)
Gradient Descent
Update (bias)
\(b := b - \alpha\,\dfrac{\partial J}{\partial b}\)
\(B := B - \alpha\,\dfrac{\partial J}{\partial B}\)
Regularization Term
\(\dfrac{\lambda}{2m}\displaystyle\sum_{j=1}^{n} w_j^2\)
\(\dfrac{\lambda}{2m}\,W^{\top}W\)
Regularization Gradient
\(\dfrac{\partial}{\partial w_j} \left( \dfrac{\lambda}{2m} \sum_{j=1}^{n} w_j^2 \right) = \dfrac{\lambda}{m} w_j\)
\(\dfrac{\partial}{\partial W} \left( \dfrac{\lambda}{2m} W^{\top}W \right) = \dfrac{\lambda}{m} W\)
