Decision Trees

Concept

A decision tree splits the data into different branches based on feature values. The tree structure consists of internal nodes (each of which represents a decision or test based on a specific feature) and leaf nodes (each of which represents the predicted outcome or class label).

Objective

The objective, used to decide which feature to split on at each node, is to decrease impurity (i.e., maximise information gain):

  • In classification, impurity is measured by entropy or Gini impurity
  • In regression, impurity is measured by variance
Variables &
Notation
\(\mathcal{I}\) Impurity function
\(v\) Index of a child node (\(v = 1, \dots, V\)), where \(V\) is the total number of child nodes produced by a split
\(S^{(v)}\) The set of training samples at node \(v\)
\(S^{\text{parent}}\) The set of training samples at the parent node, i.e., the node currently being split
\(w^{(v)} = \dfrac{|S^{(v)}|}{|S^{\text{parent}}|}\) The proportion of training samples from the parent node that flow into the child node \(v\)
Information Gain
\(\mathcal{I}(S^{\text{parent}}) - \displaystyle\sum_{v=1}^{V} w^{(v)} \cdot \mathcal{I}(S^{(v)})\)
Stopping Criteria

To prevent overfitting, the recursive splitting process is halted on the basis of one of the following criteria:

  • when a node is 100% pure (contains only one class)
  • when splitting a node will result in the tree exceeding a defined maximum depth
  • when the decrease in impurity is below a certain threshold
  • when the number of target values (samples) at a node is below a certain threshold

Tree Ensembles

Concept

A single decision tree can be highly sensitive to small changes in the training data, which often leads to high variance. This is why multiple trees, called a tree ensemble, are used.

The final prediction is the combination of the predictions of all the individual trees:

  • In classification, this combination is the majority consensus (mode) of the predicted classes
  • In regression, this combination is the mean of the predicted values
